# SKILL: Auto Template Generator

## OBJETIVO
Permitir que o sistema crie novos templates automaticamente mantendo estabilidade de PDF.

---

## REGRAS ESTRUTURAIS OBRIGATÓRIAS

1. Cada template deve usar container `.pdf-page`
2. Altura fixa: 1123px
3. Largura fixa: 794px
4. Padding superior >= 80px
5. Padding inferior >= 100px
6. Rodapé absoluto
7. Não usar CSS grid complexo
8. Não usar box-shadow pesado
9. Não usar backdrop-filter

---

## FLUXO DE CRIAÇÃO AUTOMÁTICA

1. Identificar perfil dominante do cliente
2. Criar identidade visual coerente com:
   - setor
   - ticket médio
   - tom emocional
3. Gerar novo componente React em:
   templates/AutoGenerated/
4. Registrar automaticamente no TEMPLATE_REGISTRY
5. Tornar disponível para seleção manual futura

---
